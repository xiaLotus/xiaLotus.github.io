字典特徵

字典和陣列類似，也是可變序列，但它是無序的，保存的內容是以「鍵:值」的形式存放的。
鍵是唯一的，但值可以有很多個。

特徵如下:

  1. 透過鍵而不是透過所引來讀取
  2. 無序
  3. 可變，可任意嵌套
  4. 鍵必須唯一
  5. 鍵不可變

定義字典時，每個元素都包含兩個部分–鍵、值，兩者使用「:」隔開，所有元素都放在一個大括號中
