保留字與標識符

  • 保留字

  • 標識符

  1. 由字母、下划線、數字組成，請第一個字母不得是數字
  2. 不能使用上述保留字
  3. 區分大小寫
  4. 下划線開頭的標識符具有特殊意義，應避免使用
  5. 盡量不要用中文作為標識符
